# YouTube Clone

A responsive YouTube-like interface built with **HTML and CSS** only. This project replicates the core design and layout of YouTube's homepage and video player interface.

## 📋 Project Overview

This is a frontend-only project that demonstrates web design skills by creating a pixel-perfect clone of YouTube's user interface using semantic HTML and modern CSS techniques.

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Navigation Bar** - YouTube-style header with search functionality (UI only)
- **Sidebar Menu** - Collapsible navigation menu with categories
- **Video Grid** - Organized layout displaying video thumbnails and metadata
- **Video Cards** - Individual video cards with thumbnail, title, channel name, and view count
- **Hover Effects** - Interactive hover animations for better user experience
- **CSS Grid & Flexbox** - Modern layout techniques for flexibility and responsiveness
